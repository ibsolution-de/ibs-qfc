# Features

## English

Resource Planning & Capacity Management
As a PM, I want to see team utilization heatmaps so I can instantly identify over-allocated (>100%) and under-allocated (<50%) team members across all projects.
As a Bereichsleiter, I want to see cross-project resource conflicts when the same employee is planned on multiple critical projects simultaneously, so I can make prioritization decisions during QFC calls.
As a PM, I want to allocate partial days (e.g., 50% Project A, 50% Project B) instead of full days only, to reflect realistic work splits on smaller projects.
As a PM, I want to mark vacation, sick days, and training for team members so capacity calculations automatically exclude non-working days.

Project & Financial Planning
As a Bereichsleiter, I want to see project profitability forecasts comparing planned hours vs budget, so I can identify margin risks before projects start.
As a PM, I want to link resource plans to project milestones so I can align team allocations with project phases and deliverable deadlines.
As a Bereichsleiter, I want revenue forecasts by client for next 4 quarters based on planned projects, so I can identify revenue gaps needing new business development.

Reporting & Analytics
As a PM, I want to generate monthly utilization reports by employee for client billing and internal accounting purposes.
As a Bereichsleiter, I want to compare planned vs actual capacity from historical versions to improve forecasting accuracy over time.
As a PM, I want to track project budget burn rate against allocated resources to identify projects at risk of budget overrun.

Collaboration & Workflow
As a PM, I want to share read-only views with clients so they can see planned resources without ability to modify data.
As a Bereichsleiter, I want to approve resource plans before they become official versions, with approval workflow and audit trail.
As a PM, I want to create "what-if" scenario versions to model different project combinations without affecting the official plan.

Integration & Data Management
As a PM, I want to import project budgets from SAP and export planned hours to SAP for automatic budget account creation (Factura integration).
As a PM, I want to sync with Outlook/Google calendars to reflect actual team member availability and avoid planning on blocked days.
As a Bereichsleiter, I want automated backups of all versions for compliance and audit purposes with 7-year retention.

Advanced Planning
As a Bereichsleiter, I want skill-based capacity planning to see which skills are over/under capacity and identify hiring needs.
As a PM, I want to plan by role levels (Senior/Mid/Junior) on projects to ensure proper skill mix and mentor opportunities.
As a Bereichsleiter, I want Monte Carlo simulation for opportunity planning to see probability-weighted capacity scenarios.
